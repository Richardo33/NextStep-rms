NextStep RMS is a recruitment management platform that helps HR teams manage jobs, candidates, and screening processes. It uses Supabase as the backend and supports features like candidate pipelines, resume uploads, and basic automation hooks.

## Features

- Company & HR user management  
- Job posting and candidate tracking  
- Candidate status pipeline (screening, interview, hired, rejected, etc.)  
- Resume upload and storage (PDF) via Supabase  
- Dashboard for viewing jobs and candidates  

## Tech Stack

- Next.js (React + TypeScript)  
- Supabase (Auth, Database, Storage)  
- Tailwind CSS / shadcn-ui  

## Getting Started

### Prerequisites

- Node.js LTS  
- npm or yarn  
- Supabase project (URL & anon key)  

### Installation

```bash
git clone https://github.com/Richardo33/NextStep-rms
cd nextstep-rms
